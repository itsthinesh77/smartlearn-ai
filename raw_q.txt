Easy Data Structures MCQs (with Answers)
1. Which data structure uses FIFO?
A) Stack B) Queue C) Tree D) Graph
Answer: B
2. Which data structure uses LIFO?
A) Queue B) Stack C) Array D) Linked List
Answer: B
3. Which data structure stores elements in contiguous memory?
A) Linked List B) Array C) Tree D) Graph
Answer: B
4. Which is a linear data structure?
A) Tree B) Graph C) Array D) Heap
Answer: C
5. What is the first element in a queue called?
A) Rear B) Top C) Front D) Head
Answer: C
6. What is the last element in a queue called?
A) Rear B) Front C) Top D) End
Answer: A
7. Which operation adds element to stack?
A) Pop B) Push C) Insert D) Add
Answer: B
8. Which operation removes element from stack?
A) Delete B) Remove C) Pop D) Push
Answer: C
9. Which structure uses nodes and pointers?
A) Array B) Linked List C) Stack D) Queue
Answer: B
10. Which is non-linear data structure?
A) Array B) Stack C) Queue D) Tree
Answer: D
11. What is the index of first element in array?
A) 0 B) 1 C) -1 D) Depends
Answer: A
12. Which traversal visits root first?
A) Inorder B) Preorder C) Postorder D) Level
Answer: B
13. Which traversal visits root last?
A) Inorder B) Preorder C) Postorder D) Level
Answer: C
14. Which data structure is used for recursion?
A) Queue B) Stack C) Array D) Graph
Answer: B
15. What is a node?
A) Data only B) Pointer only C) Data + pointer D) Function
Answer: C
16. Which DS is used in BFS?
A) Stack B) Queue C) Array D) Tree
Answer: B
17. Which DS is used in DFS?
A) Queue B) Stack C) Heap D) Array
Answer: B
18. Which structure has parent-child relation?
A) Array B) Stack C) Tree D) Queue
Answer: C
19. What is top of stack?
A) First element B) Last inserted C) Middle D) None
Answer: B
20. Which operation checks empty stack?
A) isEmpty B) push C) pop D) top
Answer: A
